**Agent-driven development.**

_A human sets the heading — a fleet of agents does the flying._

### How it works

```text
  human (heading)  →  agent fleet (builds)  →  multi-model panel (reviews)  →  green CI  →  ship
```

- 🛠️ Every repo has a **dedicated executor agent** doing the actual coding
- 🧭 Every PR is **cross-checked by a multi-model panel** before merge — no single-model blind spots
- 🛡️ **Branch protection + green CI** is the hard floor; nothing lands red
